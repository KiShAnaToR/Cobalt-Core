# Cobalt Core — AI-Powered Interview Platform

An intelligent mock interview platform. Upload your resume, pick a role, get AI questions and instant feedback — text or voice.

---

## 🚀 Deploy to Render (Recommended)

### One-click deploy with Blueprint

1. Push this repo to GitHub
2. Go to [render.com](https://render.com) → **New** → **Blueprint**
3. Connect your GitHub repo — Render will detect `render.yaml` automatically
4. It will create:
   - A **Web Service** (the FastAPI app)
   - A **free Postgres database**
5. You'll be prompted to fill in one env var: **`OPENAI_API_KEY`** → paste your key
6. Click **Apply** — done! You get a live URL like `https://cobalt-core.onrender.com`

> **No OpenAI key?** The app works without one — you get fallback questions and heuristic scoring. Only AI question generation, Whisper transcription, and TTS require a key.

### Manual deploy (alternative)

If you prefer not to use the Blueprint:

1. Render → **New Web Service** → connect repo
2. Set:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `uvicorn app.main:app --host 0.0.0.0 --port $PORT`
3. Add env vars:
   - `OPENAI_API_KEY` = your OpenAI key
   - `DATABASE_URL` = your Postgres URL (Render → New PostgreSQL → copy Internal URL)
   - `ENVIRONMENT` = `production`
4. Deploy

## 💻 Local Development

```bash
git clone https://github.com/YourUsername/cobalt-core.git
cd cobalt-core
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # add your OPENAI_API_KEY
python -m uvicorn app.main:app --reload
# Open http://127.0.0.1:8000
```

## Features

| Feature | Description |
|---------|-------------|
| 🤖 AI Questions | GPT-3.5 questions based on your resume & role |
| ✍️ Text Mode | Type answers, get scored feedback instantly |
| 🎥 Video Mode | Record audio → Whisper transcription → AI evaluation |
| 📊 Analysis | Vocabulary, clarity, filler words, grammar |
| 🔊 Voice Feedback | gTTS converts AI response to audio |
| 💾 History | All sessions stored in Postgres, browseable |

---

## API Reference

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/upload_resume` | Parse resume (PDF/DOCX/TXT) |
| POST | `/start_interview` | Create session, get first question |
| POST | `/submit_answer` | Submit answer, get evaluation + next question |
| POST | `/video_audio` | Submit audio, transcribe, evaluate |
| POST | `/feedback` | Analyze transcript |
| POST | `/ai_voice_response` | Get AI voice feedback |
| POST | `/end_session` | Finalise session and score |
| GET  | `/sessions` | List all sessions |
| GET  | `/sessions/{id}` | Get session details |

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `OPENAI_API_KEY` | No | Enables GPT-3.5, Whisper, TTS |
| `DATABASE_URL` | No | Defaults to SQLite locally, use Postgres in prod |
| `ENVIRONMENT` | No | Set to `production` on Render |
| `SECRET_KEY` | No | Auto-generated by Render Blueprint |

## Tech Stack

- **Backend:** Python 3.11+, FastAPI, SQLAlchemy 2.x
- **Database:** SQLite (local) / PostgreSQL (production)
- **AI:** OpenAI GPT-3.5-turbo, Whisper, gTTS
- **Frontend:** HTML5, Vanilla JS
- **Hosting:** Render
